项目下载和运行
----

- 拉取项目代码

```bash
git clone git@github.com:lanmingyi/web.git
cd  web
```

- 安装依赖
```
npm install
或 yarn install
```

- 开发模式运行
```
npm run dev 
或 yarn run dev
```

- 编译项目
```
npm run build
或 yarn run build
```
